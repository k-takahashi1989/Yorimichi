### 収益シミュレーション（月3〜5万達成ライン）

```
【広告収入】
  バナー広告 eCPM: ¥30〜80（日本市場）
  DAU 1,500人 × 3回/日表示 × eCPM ¥50 / 1000
  = ¥6,750/月

  インタースティシャル eCPM: ¥300〜800
  DAU 1,500人 × 0.5回/日 × eCPM ¥500 / 1000
  = ¥11,250/月

  広告合計: 約¥18,000/月

【サブスク収入】
  MAU 7,500人 × CVR 2% × ¥300/月
  = 150人 × ¥300 = ¥45,000/月
  Apple/Google 手数料(15-30%) 差引後: ¥31,500〜38,250/月

  → 現実的には CVR 1〜1.5% を想定
  → 75〜112人 × ¥300 = ¥22,500〜33,750（手数料前）

【合算目安】
  広告 ¥15,000〜18,000 + サブスク ¥16,000〜28,000
  = 月額 ¥31,000〜46,000 ✅
```

**結論: MAU 5,000〜7,500人、サブスク CVR 1〜2% で月3〜5万円は達成可能**

## リスクと対策

| リスク | 影響 | 対策 |
|--------|------|------|
| ユーザー獲得が想定より遅い | 収益目標未達 | ASO改善サイクルを2週間毎に回す。広告出稿（月¥5,000程度）も検討 |
| サブスク CVR が1%未満 | サブスク収入不足 | ペイウォール A/Bテスト、価格帯テスト、機能差別化の強化 |
| Firebase 無料枠超過 | コスト発生 | Blaze プラン移行（従量課金）。MAU 5,000程度なら月¥500〜1,000程度 |
| 競合アプリの台頭 | シェア低下 | 「位置×買い物リスト」のニッチ特化を維持。ジオフェンスの精度・UXで差別化 |
| Android のみでユーザー母数が不足 | MAU伸び悩み | 海外展開（英語・韓国語・中国語繁体）で Android シェアの大きい市場を狙う |
