# Yorimichi 使い方ガイド

場所に着いたら買い物リストを通知してくれるリマインダーアプリです。

---

## 基本的な使い方

### 1. メモを作成する

1. ホーム画面の右上にある **＋ボタン** をタップ
2. 場所を選択する画面が表示されます（スキップも可能）
3. メモのタイトルを入力（例：「スーパーで買うもの」）
4. チェックリストにアイテムを追加
5. 「確認する」をタップして保存

### 2. 場所を登録する

1. メモ詳細画面の「場所」セクションで **追加** をタップ
2. 地図上で長押ししてピンを配置、または検索バーで場所を検索
3. 場所の名前を入力（例：「スーパー三和」）
4. 通知半径をスライダーで調整（100m〜1000m）
5. 通知タイミングを選択（到着時 / 出発時）
6. 「この場所を保存」をタップ

### 3. 通知を受け取る

1. メモ詳細画面のベルアイコンをタップして通知を **ON** に
2. 設定画面で「リマインドを開始する」をタップ
3. 登録した場所に近づくと自動的に通知が届きます

> 通知を受け取るには **位置情報（常に許可）** と **通知の許可** が必要です。設定画面の「位置情報・通知をオンにする」ボタンから一括設定できます。

## 便利な機能

### チェックリスト操作

- **チェック / チェック解除**: アイテムをタップ
- **一括チェック / 解除**: チェックリスト右上のアイコンをタップ
- **完了済みを非表示**: 目のアイコンをタップすると、チェック済みアイテムを隠せます
- **並び替え**: 編集画面でアイテム左の上下矢印をタップ

### 期限設定

- 編集画面のカレンダーアイコンから期限日を設定
- 期限当日に通知が届きます
- メモ詳細画面に期限バッジが表示されます（期限切れは赤色で表示）

### メモ（補足ノート）

- 編集画面の「メモ」欄にフリーテキストで補足情報を記入できます

### 通知時間帯の設定

- 設定画面の「通知時間帯」で有効にすると、指定した時間帯のみ通知が届きます
- 深夜や早朝の通知を防ぎたいときに便利です

### 通知モード

- メモ詳細画面のベルアイコンを長押しすると通知モードを選択できます
  - **サイレント**: 音・バイブなし（通知バーのみ）
  - **通知（標準）**: 音・バイブあり

## 共有機能

家族や友人とメモを共同編集できます。

### メモを共有する

1. メモ詳細画面の **共有ボタン** をタップ
2. 共有コードが生成されます
3. 共有コードを相手に送信

### 共有メモをインポートする

1. ホーム画面右上のメニューから「共有コードでインポート」をタップ
2. 受け取った共有コードを入力
3. メモが自分のリストに追加されます

### 共有メモの同期

- 共有メモの最新状態を取得するには **同期ボタン**（回転矢印）をタップ
- プレミアムプランでは、メモ詳細画面を開いている間リアルタイムで自動同期されます

### 更新通知（プレミアム）

- 共有メモの内容を更新したら「共有相手に更新を通知」ボタンでプッシュ通知を送れます

## プレミアムプラン

プレミアムにアップグレードすると、以下の機能が利用できます：

| 機能 | 無料 | プレミアム |
|------|------|-----------|
| メモ数 | 5件 | 無制限 |
| アイテム数/メモ | 10個 | 無制限 |
| 登録地点数/メモ | 2か所 | 10か所 |
| 共有人数 | 1対1 | 複数人 |
| 共有メモ同期 | 手動 | リアルタイム |
| クラウドバックアップ | なし | 日次自動 |
| 広告 | あり | なし |

### クラウドバックアップ（プレミアム）

- アプリ起動時に自動でクラウドにバックアップ
- 設定画面から手動バックアップ・復元も可能
- 端末の故障や機種変更時にデータを復元できます

## よくある質問（FAQ）

### Q: 通知が届かない

以下を確認してください：

1. メモの通知がON（ベルアイコンが緑色）になっているか
2. 設定画面で「リマインドを開始する」が有効か
3. 端末の位置情報が「常に許可」になっているか
4. 端末の通知設定でアプリの通知が許可されているか
5. 通知時間帯の設定で現在時刻が範囲内か

### Q: バッテリー消費が増えた

位置連動リマインド機能はバックグラウンドで位置情報を使用するため、バッテリー消費が若干増える場合があります。使わないときは設定画面から「リマインドを停止する」ことで消費を抑えられます。

### Q: メモを削除したい

ホーム画面でメモを左にスワイプすると削除ボタンが表示されます。

### Q: 共有メモの場所を変更できない

共有メモの場所はオーナー（共有した人）のみ編集できます。コラボレーター（共有を受けた人）は場所の閲覧のみ可能です。

### Q: データを移行したい

プレミアムプランのクラウドバックアップ機能を使うと、新しい端末にデータを復元できます。設定画面 > プレミアムプラン > クラウドバックアップから操作してください。
