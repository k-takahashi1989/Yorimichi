Yorimichi — Canva Hero Template

Files:
- canva_hero_template_1920x1080.svg (1920x1080 SVG)

Usage:
1. Open Canva and go to "Uploads" → "Upload files" then upload the SVG file.
2. Drag the SVG onto a blank canvas. Canva will import the vector shapes; you can edit text and colors.
3. Replace the phone screen placeholder with your own screenshot:
   - Select the rectangle with id "screen_placeholder" (in Canva it may appear as a rectangle). Delete it.
   - Upload your device screenshot and drag it into the phone mockup. Resize and mask as needed.

Recommended export sizes:
- Google Play / Feature hero (wide): 1024 x 500 px (or 1920 x 1080 px for ads)
- High-res store image: 1242 x 2688 px (if you want very high DPI)

Fonts:
- Use rounded-friendly JP fonts: "M PLUS Rounded 1c" or "Noto Sans JP" (Canva has similar rounded fonts).

Colors:
- Teal (main): #00AFAF
- Accent (CTA): #FF8A3D

Notes:
- The SVG contains a dashed-stroke rectangle named "screen_placeholder". Keep this area clear before exporting so you can insert real screenshots.
- If Canva flattens text on import, you can re-create text elements in Canva using recommended fonts.
